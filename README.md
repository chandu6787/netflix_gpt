# 🎬 Netflix GPT

A Netflix-inspired streaming app built with React, Redux, Firebase, and powered by Groq AI for smart movie search.

🔗 **Live Demo:** [netflix-gpt-eight-beta.vercel.app](https://netflix-gpt-eight-beta.vercel.app)

---

## ✨ Features

- 🔐 Authentication (Sign Up / Sign In / Sign Out) via Firebase
- 🎥 Browse Now Playing, Popular, and Top Rated movies
- 🤖 Groq AI-powered smart movie search
- 🌐 Multi-language support
- 📱 Fully responsive design with Tailwind CSS

---

## 🛠️ Tech Stack

- **Frontend:** React, Tailwind CSS
- **State Management:** Redux Toolkit
- **Authentication & DB:** Firebase
- **Movie Data:** TMDB API
- **AI Search:** Groq API

---

## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/chandu6787/netflix_gpt.git
cd netflix_gpt
```

### 2. Install dependencies
```bash
npm install
```

### 3. Set up environment variables

Create a `.env` file in the root:
```env
REACT_APP_TMDB_KEY=your_tmdb_api_key
REACT_APP_GROQ_KEY=your_groq_api_key
```

### 4. Run the app
```bash
npm start
```
